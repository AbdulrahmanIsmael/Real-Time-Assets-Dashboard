# Investment Portfolio Dashboard

A dashboard to view, control, sort or filter your assets, helping you to make critical decistion about your investments and your life

## Features

- Table to view all of your assets with details
- Sort your assets
- Filter your assets
- Search for assets by name

---

## Techs & Tools

- Next.js
- Tailwind CSS

---

## Setup and Run

1. Install dependencies `npm install`
2. Build the product `next build`
3. Serve the production for testing `next start`
